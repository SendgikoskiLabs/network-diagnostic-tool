# Network Diagnostic Tool

A comprehensive Python tool for network diagnostics including Ping, Traceroute, and Latency measurements.

## Features

- **Ping**: Classic ICMP ping with packet loss and latency statistics
- **Traceroute**: Trace the route to a host with hop-by-hop information
- **Latency Check**: TCP-based latency measurements to specific ports
- **All-in-One**: Run all diagnostics at once
- **Cross-Platform**: Works on Windows, Linux, and macOS
- **JSON Output**: Export results in JSON format
- **Statistical Analysis**: Provides min, max, average, and standard deviation

## Requirements

- Python 3.7+
- Administrative/root privileges for ping and traceroute operations
